======================================================================
INTENT-TO-COURSE MAPPING
======================================================================

“My child has never coded.”
Ask age, grade, and interest.
Likely options:
- Scratch Level 1
- Beginner Online Python for age 8–12 if ready for text coding
- Web Development Level 1
- Block-Based App Making

“My child loves games.”
Ask whether they want to build games and whether they have coded.
Likely options:
- Scratch Level 1
- Scratch Level 2
- Python Game Development custom course
- Roblox or Minecraft custom course

“My child wants a mobile app.”
- Intro to Apps Coding
- Custom text-based app course after assessment

“My child wants a website.”
- Beginner: Web Level 1
- Knows HTML/CSS: Web Level 2
- Knows JavaScript: React or Full-Stack custom pathway

“My child wants Python.”
- Age 8–12 beginner: Beginner Online Python
- Visual learner: Beginner Python with Graphics
- Knows basics: Intermediate Python custom course
- Wants games: Python Game Development
- Wants AI/data: Python for AI or Data Science custom pathway

“My child wants AI.”
- Grades 6–8: AI for Coding 6th–8th
- Grades 8–12: AI for Coding 8th–12th
- Grade 8: assess experience and project maturity
- Younger learner with Scratch experience: Scratch Level 2 AI-themed projects
- Specific AI goal: custom Prompt Engineering, AI Project, or Python for AI

“My child wants ChatGPT.”
Clarify:
- Prompt writing and responsible use
- Coding support
- Building an AI project
- Java/SQL project work
