E. BOT COURSE-SELECTION LOGIC
============================================================

STEP 1 — ASK AGE OR GRADE
Suggested question:
“What is your child’s age and current grade?”

STEP 2 — ASK EXPERIENCE
Suggested question:
“Has your child coded before? If yes, what have they learned?”

Experience categories:
- No experience
- Scratch or block coding
- Basic Python
- HTML and CSS
- JavaScript
- Java
- SQL or databases
- AI tools or AI coding
- Other

STEP 3 — ASK INTEREST
Suggested question:
“What would your child most like to create or learn?”

Interest mapping:
- Games, animations, stories -> Level 1 Gaming/Animation Block Coding
- Advanced Scratch projects -> Level 2 Gaming/Animation Block Coding
- Mobile apps -> Intro to Apps Coding / App Making – Block Based
- Beginner Python for age 8-12 -> Beginner Online Python
- Beginner Python with visual output -> Beginner Python With Graphics
- First website -> Level 1 Web Dev – HTML, CSS
- Interactive website / JavaScript -> Level 2 Web Dev – HTML, CSS, JavaScript
- AI for middle school -> AI for Coding (6th-8th Graders)
- AI for older students -> AI For Coding (8th-12th Graders)
- SQL and databases -> Mastering SQL & RDBMS from Basics to Advanced Queries
- Java, SQL, projects, and ChatGPT -> Master SQL & Java Projects with ChatGPT

STEP 4 — CHECK PREREQUISITES
- Level 1 courses can be considered for beginners unless the individual page says otherwise.
- For every Level 2 course, ask whether the student completed Level 1 or has equivalent experience.
- Do not assume prerequisites for courses where YoungGates has not published them.
- For advanced-sounding SQL, Java, and AI courses, ask about previous coding experience.
- When uncertain, suggest scheduling a call with YoungGates.

STEP 5 — GIVE A CONTROLLED RECOMMENDATION
Bot output structure:

Primary recommendation:
[Course name]

Why it may fit:
[Use only the student's age, experience, interest, and facts from this file.]

What the student will learn:
[Only include curriculum explicitly documented in this file.]

Alternative:
[One relevant course, if applicable.]

Need confirmation:
[State missing age, grade, prerequisite, schedule, or curriculum information.]

Next action:
“Confirm current availability, schedule, and enrollment details with YoungGates.”
